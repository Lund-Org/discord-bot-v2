# Nouvelles fonctionnalités Avril 2023
</Center>

## Jeux attendus

La fonctionnalité était teasée dans le menu, la voici la voilà.
Vous pouvez la trouver si vous vous authentifier, dans le menu en cliquant en haut à droite sur votre profil.
C'est très simple : 
- Vous cherchez un jeu
- Vous sélectionnez la plate-forme qui vous intéresse
- Vous dites si vous voulez que le jeu soit ajouté au backlog à sa sortie
- Vous êtes alertés le jour de la sortie sur Discord !

Si jamais un jeu n'a pas encore de date de sortie, ce n'est pas grave.
Par exemple, si vous cherchez Final Fantasy VII Rebirth, il sera marqué comme sortant le 31 Décembre 2023.
Mais le jour où la date sera annoncé, votre jeu attendu sera mis à jour automatiquement.
Cette fonctionnalité est pratique pour surveiller des jeux et avoir des mises à jour sur sa date de sortie, sans avoir à s'embéter, sans avoir à ajouter le jeu à sa wishlist.

Vous pouvez visualiser vos jeux attendus sous la forme d'une liste ou d'un calendrier.
Sur les profils des utilisateurs vous pouvez voir les jeux attendus, en format liste uniquement.


## Backlog

Cette fonctionnalité est déjà sortie il y a plusieurs mois mais il est bon de rappeler qu'elle existe.
Là aussi dans le menu vous pouvez trouver la fonctionnalité.
Elle vous permet de marquer les jeux que vous voulez faire/que vous faites/que vous avez fait.
Vous pouvez également, si vous avez fini ou abandonné un jeu, le noter et donner faire une remarque.
Toutes ces mises à jour (statut & notation/remarque) sont partagées sur Discord


## Liste des utilisateurs

Depuis la dernière mise à jour, il est possible pour les gens d'avoir un "compte" juste en étant sur le serveur Discord.
De ce fait, les utilisateurs peuvent avoir un backlog ou attendre des jeux, sans jouer au gacha (qui était l'unique manière d'avoir un "compte" avant).
Le problème, c'est qu'il n'y avait aucun moyen d'aller trouver ces profils et pour les joueurs du gacha il faut passer par le classement.
Maintenant dans le menu, vous avez une entrée pour voir tous les utilisateurs, triés par ordre alphabétique, pour facilement voir les profil gacha/backlogs/jeux attendus des gens.


## Gacha

Pour finir, dernier ajout sur le gacha, la commande `/gacha warn`.
Cette commande permet de mettre en place un avertissement quand vous dépassez les 13 000 points.
Pour rappel, vous ne pouvez pas accumuler plus de 15 000 points.
Lorsque vous posterez un message qui vous fait dépasser les 13 000 points, Maurice vous préviendra.
Il y a un cooldown de 6h, donc si 6h plus tard vous n'avez toujours pas dépensé vos points et que vous repostez un nouveau message, vous serez de nouveau alertés.

<p style={{ textIndent: 0 }}>*Pour les non-anglophones, lorsque vous écrivez la commande, vous avez une option et vous devez choisir True ou False. True veut dire "vrai", donc vous voulez être alerté, et False veut dire "faux", donc ne pas/plus être alerté. Ce n'est pas mon choix personnel, c'est la faute de Discord.*</p>
